# Tablero-de-ingresos

Para la creación de este tablero en Power BI, se llevaron a cabo las siguientes etapas clave:

### Recolección y Organización de Datos
- Se recopiló la información relacionada con los ingresos de la empresa X, provenientes de la venta de diferentes productos por franquicia y subfranquicia.
- Los datos fueron extraídos desde diversas fuentes en formato excel y csv.

### Limpieza y Transformación de Datos con Power Query
- Se realizó una depuración de la información, eliminando valores nulos, inconsistencias y registros duplicados.
- Se aplicaron transformaciones en Power Query, como la normalización de nombres, conversión de tipos de datos y creación de columnas calculadas para mejorar la estructura de los datos.

### Modelado de Datos en Power BI
- Se establecieron relaciones entre tablas para asegurar una correcta conexión de los datos.

### Diseño de Visualizaciones Interactivas
- Se incluyeron filtros y segmentaciones para facilitar la exploración de los datos según diferentes criterios.

### Monitoreo y Actualización Continua
- Se configuraron actualizaciones automáticas para mantener los datos al día.
- Este proceso garantiza que el tablero proporcione una visión clara y estratégica de los ingresos de la empresa, facilitando la toma de decisiones basada en datos precisos y actualizados.
